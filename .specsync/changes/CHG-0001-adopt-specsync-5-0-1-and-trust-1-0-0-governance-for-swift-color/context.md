# Context

Swift Color is a Swift 6 library covering RGB, HSL, HSV, LAB/LCH, named colors, palettes, manipulation, and accessibility. It is verified on Linux and macOS and publishes DocC through Pages. It has no prior SpecSync companions or committed threshold.
